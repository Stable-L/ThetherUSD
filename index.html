Web3 Integration

Designed for easy integration with TronLink and TronWeb. The UI keeps on-chain values visible, uses meaningful naming for token metadata and prepares places for proof of reserve / audit badges.

How to use

  1. Install TronLink extension and unlock your wallet.
  2. Click Connect Wallet and allow the dApp to read your address.
  3. Check your token balance and send transfers securely via the wallet popup.
